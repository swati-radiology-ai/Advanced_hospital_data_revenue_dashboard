# Advanced_hospital_data_revenue_dashboard
Advanced excel dashboard for hospital revenue analysis with KPI,patient trends,and financial insights.
# Project overview 
This project is an advanced excel dashboard designed for hospital revenue analysis.it helps in 
tracking financial performance, patient trends, and key performance indicators (KPI).
## Features
- Total Revenue KPI
- Monthly revenue trend
- Patient admission analysis
- department-wise revenue
- Payment mode analysis
- Completed vs pending status tracking
## Tools used
- Microsoft excel
- pivot tables
- charts & graphs
- Data cleaning techniques
## Dataset
the dataset includes:
- Patient name
- age & gender
- department mode
- status(completed/pending)
## insights
- identifies top revenue - generating departments
- tracks monthly growth trends
- helps in hospital decision - making
## purpose
this project is built for learning and showcasing data analysis skills for healthcare and hospital management.
